## 关于我们

沪漫工厂是一个 AI 动漫创作平台，致力于降低动漫创作门槛，让更多中国好故事被看见。

### 我们的理念

- 让创作回归创作本身：更低门槛、更简工具、更加纯粹
- 让更多中国好故事被看见
